# POS System

A full-stack Point of Sale (POS) and supermarket management system built using Spring Boot, React, Vite, and MySQL.

---

## Features

- User Authentication with JWT
- Role-Based Access Control
- Product Management
- Inventory Tracking
- Sales Processing
- Receipt Generation
- Sales Reports
- Restock History
- Protected Routes
- Password Reset Functionality
- Barcode Scanner Support
- Thermal Receipt Printing

---

## Technologies Used

### Backend
- Java 23
- Spring Boot
- Spring Security
- JWT Authentication
- MySQL
- Maven

### Frontend
- React
- Vite
- Axios
- CSS

---

## Project Structure

```text
frontend/     -> React + Vite frontend
possystem/    -> Spring Boot backend
```

---

## Installation & Setup

### Clone Repository

```bash
git clone https://github.com/BushMat55/POS-System_Abihub.git
```

---

## Backend Setup

```bash
cd possystem
mvn spring-boot:run
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
